CMS 后台

从这里进入写作后台

后台由 Pages CMS 提供，内容会直接保存到 GitHub 仓库，再由 GitHub Actions 自动发布到博客。

里面有少量系统按钮还是英文，但文章、日记、About、待办和字段说明已经尽量中文化。

进入 Pages CMS 返回博客首页

只有拥有仓库写入权限的 GitHub 账号才能保存和发布内容。

写作提示

  • 正式文章点击左侧 文章，新建或编辑长一点的内容。
  • 日记点击左侧 日记，写短记录，路径仍然是 /notes/。
  • 文件名新建文章或日记时，用英文短横线并保留 .md，例如 quiet-note.md。
  • 系列系列文章填写 系列 和 系列顺序，普通文章留空。
